# Pizza Sales Analysis Dashboard
This is a Power BI-based Pizza Sales Dashboard that analyzes pizza sales data to provide meaningful business insights through interactive visualizations. 
The dashboard tracks key performance indicators such as total revenue, total orders, pizzas sold, and average order value, while also analyzing sales by pizza category, size, time, and date. 
It helps identify top- and low-performing pizzas, peak ordering hours, and customer purchasing trends, enabling data-driven business decisions. 
This project demonstrates the use of Power BI, Power Query (ETL), DAX, Data Modeling, Data Cleaning, Interactive Dashboards, Charts, KPIs, Filters/Slicers, and Data Visualization to transform raw sales data into actionable insights.
